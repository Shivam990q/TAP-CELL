# Job Descriptions

Company-wise organized. Har company ka apna subfolder hai:

```
06-job-descriptions/
├── infosys/            → infosys-specialist-programmer-2027.md
├── josh-technology/    → josh-technology-frontend-2027.md, josh-technology-DEEP-RESEARCH.md
└── stackmentalist/     → stackmentalist-python-fullstack-2027.md
```

- Nayi company aaye to `06-job-descriptions/<company>/` folder banao, JD `<company>-<role>.md` naam se save karo.
- Corresponding prep/application files `05-applications/<company>/` me rehte hain.
- Main isse ATS-tuned tailored resume banata hoon master data se.
