## Problem
A Noida-based consultancy needed a professional digital presence with self-serve content
management (case studies, expertise, leads) without depending on developers for updates.

## Tech stack
- Frontend: React + TypeScript, Vite, Tailwind CSS + shadcn/ui, Framer Motion, TanStack
  Query (+ Priority Hints), React Router, Lenis
- Backend: Django 4.2.7 + Django REST Framework
- DB: PostgreSQL
- Infra/Deploy: Gunicorn + Nginx, CI/CD to VPS with SSL, automated zero-downtime deploys

## What I built (key features)
- Custom CMS panel with full CRUD for complete content control
- Dual templates for flexible case-study layouts; JSON-based case studies
- Contact inquiry system with lead tracking
- Industry-wise expertise management
- Mobile-first responsive UI; SEO-ready (meta + OG + Twitter Cards)
- Speed optimization: lazy loading, smart fetch, image/media optimization
