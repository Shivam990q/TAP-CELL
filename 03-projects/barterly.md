# Barterly (Redesigned) — Skill-trading Platform

- One-liner: A platform to trade skills/services without money, rebuilt from scratch.
- Status: Live + open-source
- Links: Live: (from LinkedIn post) | GitHub: https://github.com/Shivam990q/Barterly-Redesigned
- Timeline: ~1 year ago
- Role: Solo — redesigned and rebuilt from scratch (original concept by Kartik Sharma & team)

## Problem
Enable people to exchange skills directly (barter) instead of using currency, with a modern,
responsive, intuitive interface.

## Tech stack
- Frontend: React + TypeScript, Vite, Tailwind CSS + shadcn/ui
- State/Data: React Query; Routing: React Router DOM
- Forms: React Hook Form + Zod; Animation: Framer Motion
- UI: Radix UI primitives, Lucide icons, date-fns, Recharts, Sonner

## What I built (key features)
- Skill Wallet (track skill hours / transactions); Skill Marketplace
- Live sessions with integrated video conferencing; barter (currency-free) exchange
- AI-powered recommendations + custom AI-powered search in marketplace
- User profiles, reviews, session scheduling; auth (signup/login); full responsiveness
